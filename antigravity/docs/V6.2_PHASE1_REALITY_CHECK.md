## ⚠️ Critical Issues Identified

### 1. Patterns Quá Generic (90% systems fail here)

**Current patterns:**
- `syntax_error`
- `missing_import`
- `no_op_patch`
- etc.

**Problem:** LLM đã biết những cái này → inject vào prompt = **không thêm information mới**

**Evidence:**
```
Avoid: missing import
Prefer: add import statement
```
→ LLM: "ok noted bro" … rồi vẫn làm sai 😄

---

### 2. Prompt Injection Quá "Hiền"

**Current format:**
```
1. Pattern: missing import (import_missing)
   ❌ AVOID: adding new symbol without checking imports
   ✅ PREFER: add import statement before using new symbol
   Confidence: 80%
```

**Problem:** Không đủ **directive** và **cụ thể**

**What LLM needs:**
```
[CRITICAL FAILURE PATTERN #1]
Previous attempt: Added null check for "TypeError: undefined is not a function"
Why it failed: Root cause was missing import, not null value
What you MUST do differently:
  1. When you see "undefined is not a function" → CHECK IMPORTS FIRST
  2. Do NOT add defensive null checks as first response
  3. Trace the function name in error to import statements
Confidence: 90% (seen 9 times)
```

---

### 3. Missing Context Awareness

**Current pattern:**
```python
{
    "type": "missing_import",
    "cause": "missing import",
    "symbols": ["useState"]
}
```

**What we need:**
```python
{
    "type": "wrong_fix_strategy",
    "cause": "added null check for undefined function error",
    "context": {
        "error_type": "TypeError: undefined is not a function",
        "file_type": "react_component",
        "framework": "react"
    },
    "attempted_fix": "added null check before function call",
    "correct_direction": "fix import statement - function not imported"
}
```

**Difference:** Context + attempted_fix + correct_direction = **actionable insight**

---

### 4. No Feedback Loop

**Current flow:**
```
Record failure ✅
Inject into prompt ✅
```

**Missing:**
```
Measure if lesson helped ❌
Track effectiveness ❌
Prune low-signal patterns ❌
```

**Need to add:**
```python
pattern.times_injected = 10
pattern.times_helped = 7
pattern.effectiveness = 0.7  # 70% success rate
```

→ Only inject patterns with effectiveness > 0.5

---

### 5. Retrieval May Miss Real Signal

**Current scoring:**
- Pattern type match: 40%
- Keyword match: 30%
- Symbol overlap: 20%
- Recency: 10%

**Problem:** Keyword matching dễ miss pattern thật

**Better scoring:**
- **Effectiveness: 40%** (only high-signal patterns)
- **Context match: 30%** (file_type, framework, error_type)
- Pattern type: 20%
- Frequency: 10%
