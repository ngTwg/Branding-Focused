# 💻 Skill: Production Code Generation

> **PURPOSE:** Architecting and implementing clean, scalable, and production-ready code modules.
> **CATEGORY:** Backend / Frontend
> **TIER:** 2+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `code`, `implement`, `build`, `architecture`, `refactor`.
- Goal: Generating full modules, APIs, or complex algorithms from requirements.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Requirements Extraction**: Parse the query for functional constraints, edge cases, and tech stack details.
2. **Architecture Sketch**: Propose **3 implementation options** (e.g., Simple vs. Modular vs. Performance-optimized).
3. **Implementation**: Generate the full code block with built-in error handling and documentation.
4. **Validation Logic**: Create test cases and define edge case behaviors.
5. **Deployment Notes**: Provide context on environment variables, dependencies, and scaling.

---

## 📝 OUTPUT SIGNATURE

- **Architecture Diagram**: Markdown-based or Mermaid flow.
- **Code Blocks**: Clean, documented code with version-specific syntax.
- **Tests**: Unit or integration test snippets.
- **Scale Notes**: Performance considerations and limitations.

---

## 🧪 BENCHMARK TASK

- **Input**: "Viết scalable REST API cho user auth"
- **Output**: Three architecture variants -> Express.js implementation -> JWT logic -> Test cases -> Scale notes.
