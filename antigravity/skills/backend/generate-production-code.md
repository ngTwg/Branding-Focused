# 🚀 Skill: Production Code Generation

> **PURPOSE:** Writing robust, scalable, and secure application logic.
> **CATEGORY:** Backend
> **TIER:** 1+

---

## 🚦 ACTIVATION SIGNALS
- Keywords: `implement`, `build`, `write API`, `create service`, `generate code`.
- Task: Implementing business logic, data models, or API endpoints.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Requirement Extraction**: Functional (what it does) vs. Non-functional (perf, security, scale).
2. **Architecture Sketch**: Draft 3 design options mentally (or Mermaid) for complex logic.
3. **Robust Implementation**: Inclusion of input validation (Zod, Pydantic), error handling, and types (TS, Python typing).
4. **Test Generation**: Happy path, edge cases, and failure mode tests.
5. **Deployment Readiness**: Environment configuration validation, logging, and documentation markers.

---

## 📝 OUTPUT SIGNATURE
- Clean, modular code blocks.
- Inline comments summarizing non-obvious logic.
- Associated unit tests.
- Brief "How-to-Run" section.

---

## 🧪 BENCHMARK TASK
- **Input**: "Build a scalable user authentication API with JWT."
- **Output**: Controller -> Auth Service -> User DB Model -> Router -> Tests.
