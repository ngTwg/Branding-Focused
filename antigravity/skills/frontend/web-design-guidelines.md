# 💎 Skill: Web Design Guidelines

> **PURPOSE:** Design principles and aesthetics for premium, production-grade web interfaces.
> **CATEGORY:** Frontend
> **TIER:** 1+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `aesthetic`, `UI pass`, `premium design`, `glassmorphism`, `dark mode`.
- Goal: Moving from "functional" to "visually stunning" (Wowed at first glance).

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify brand personality (Modern, Minimalist, Bold).
2. **Typography Setup**: Systematize fonts (Inter/Outfit), weights, and letter-spacing (`-0.02em`).
3. **Color System**: Define HSL-based palettes for consistency and easy dark-mode generation.
4. **Layout Principles**: Apply 12-column grid, generous whitespace (whitespace is your friend!), and F-pattern reading.
5. **Interactive Feedback**: Implement subtle hover and active states (scale 0.98), micro-animations, and grain/noise textures.
6. **Efficiency Pass**: Remove generic "browser-default" elements (borders, shadows, inputs).

---

## 📝 OUTPUT SIGNATURE

- Design strategy proposals.
- Styling tokens (CSS/Tailwind).
- UI mockups/implementations.

---

## 🧪 BENCHMARK TASK

- **Input**: "Make this basic table look like a premium SaaS dashboard."
- **Output**: Glassmorphism (`backdrop-blur-xl`) -> Shadow-sm + Border-white/10 -> HSL accent colors -> -0.01em letter spacing.
