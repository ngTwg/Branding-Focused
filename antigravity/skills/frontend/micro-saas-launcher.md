# 🚀 Skill: Micro SaaS Launcher

> **PURPOSE:** Efficiently building and launching minimal, focused, and high-performance SaaS applications.
> **CATEGORY:** Frontend
> **TIER:** 2+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `SaaS`, `Micro-SaaS`, `MVP`, `subscription app`.
- Goal: Launching a functional value-driven app in < 1-2 weeks.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify core value proposition (Problem -> Solution) and target customer.
2. **Tech Selection**: Use "Ship Fast" stacks: Next.js + Tailwind + Clerk (Auth) + Stripe (Payments) + Supabase (DB).
3. **Core Feature Build**: Implement ONLY the essential product behavior (MVP).
4. **Auth & Account Handling**: Add login, profile, and basic settings with Clerk or manual auth.
5. **Subscription Billing**: Integrate Stripe Checkout/Portal for monthly/annual pricing.
6. **Launch UI Pass**: Standard SaaS layout (Dashboard, Landing, Settings, Support).

---

## 📝 OUTPUT SIGNATURE

- Functional Micro SaaS (React/Next).
- Stripe & Clerk integrations.
- Minimalist landing page + dashboard.

---

## 🧪 BENCHMARK TASK

- **Input**: "Launch a simple AI image-captioning Micro SaaS."
- **Output**: Next.js app -> Clerk auth -> Replicate API integration -> Stripe pricing table.
