# 🌬️ Skill: Tailwind Patterns

> **PURPOSE:** Scalable, utility-first CSS (Tailwind) for rapid and consistent UI development.
> **CATEGORY:** Frontend
> **TIER:** 1+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `TailwindCSS`, `utility-first`, `design token`, `spacing system`.
- Task: Styling new components, implementing dark mode, or setting up responsive layouts.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify required layout metrics (e.g., `flex`, `grid`, `space-y`).
2. **Design Token Mapping**: Define colors, fonts, and spacing in `tailwind.config.ts`.
3. **Responsive Design**: Implement `sm:`, `md:`, `lg:`, `xl:` breakpoints for mobile-first views.
4. **Interactive States**: Use `hover:`, `focus:`, `active:`, `group-hover:` for feedback.
5. **Dark Mode Integration**: Implement `dark:` variants for a premium dark-mode experience.
6. **Efficiency Pass**: Extract common patterns into classes/components to avoid "class soup."

---

## 📝 OUTPUT SIGNATURE

- Tailwind-styled TSX/HTML files.
- `tailwind.config.ts` configurations.
- Custom utility plugins.

---

- **Input**: "Build a responsive, dark-mode-ready nav bar with Tailwind."
