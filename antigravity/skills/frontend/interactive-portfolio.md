# 🎨 Skill: Interactive Portfolio

> **PURPOSE:** Design and build high-impact, story-driven personal or agency portfolios.
> **CATEGORY:** Frontend
> **TIER:** 2+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `portfolio website`, `interactive resume`, `personal site`.
- Goal: Showcasing projects with visual excellence and smooth interactions.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify target audience, key projects, and core message (e.g., "Minimalist", "Creative", "Tech-heavy").
2. **Visual Hierarchy**: Create sections (Hero, Projects, Skills, About, Contact) with a clear flow.
3. **Interactive Project Showcase**: Use animated cards, horizontal scroll, or 3D previews for case studies.
4. **Smooth Transitions**: Implement page transitions with `framer-motion` or `GSAP`.
5. **Aesthetics Pass**: Apply premium typography (Inter/Outfit), HSL color palettes, and glassmorphism.
6. **Performance Optimization**: Ensure fast image loading (Next/Image) and low LCP (Largest Contentful Paint).

---

## 📝 OUTPUT SIGNATURE

- Complete Portfolio artifacts (React/Next/HTML).
- Interactive project galleries.
- Contact form implementation.

- **Input**: "Build a high-end designer portfolio with a project slider."
