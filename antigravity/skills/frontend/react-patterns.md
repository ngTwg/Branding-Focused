# ⚛️ Skill: React & UI Patterns

> **PURPOSE:** Clean, reusable, and high-performance React and UI development with TypeScript.
> **CATEGORY:** Frontend
> **TIER:** 1+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `React`, `TypeScript`, `Next.js`, `UI design`, `component architecture`.
- Task: Building new features, refactoring components, or implementing complex UI logic.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify data models, state requirements, and view logic.
2. **Component Separation**: Functional vs. Presentational, Smart vs. Dumb components.
3. **State Management**: Choose: `useState` (local), `useContext` (shared), `useReducer` (complex), or Atomic (jotai).
4. **Pattern Application**: React Hooks (Custom), HOCs (Legacy), Compound Components, Context API.
5. **Modern UI Aesthetics Pass**: Apply: Glassmorphism (`backdrop-blur`), HSL colors, Hover micro-interactions.
6. **Efficiency Verification**: Use `React.memo`, `useMemo`, and `useCallback` to prevent unnecessary re-renders.

---

## 📝 OUTPUT SIGNATURE

- Scalable React component tree (TSX).
- Custom hooks for modular logic.
- Responsive, interactive UI layouts.

---

- **Input**: "Build a complex form with validation and multi-step progress bar."
