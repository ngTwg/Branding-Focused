# 🤖 Skill: Orchestrate Agent Workflows

> **PURPOSE:** Strategic multi-agent planning and coordination.
> **CATEGORY:** Deep Tech / AI Agents
> **TIER:** 3-4

---

## 🚦 ACTIVATION SIGNALS
- Keywords: `multi-agent`, `coordinator`, `orchestrate`, `swarm`, `agentic workflow`.
- Task: Complex planning involving multiple specialized personas.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Sub-Task Decomposition**: Break down the task into atomic actions for specialized agents.
2. **Dependency Mapping**: Enumerate predecessors and successors for every action.
3. **Agent Selection**: Assign tasks to specific agents/tools based on capability scores.
4. **Data Sync Strategy**: Define how state/artifacts are passed between agents.
5. **Success Definition**: Set clear criteria for when the workflow is complete and valid.
6. **Error Correction**: Implement "Circuit Breakers" for agent failure loops.

---

## 📝 OUTPUT SIGNATURE
- Numbered execution plan.
- Checkpoint questions for user validation.
- Explicit assumptions upfront.
- (Optional) Sequential list of tool/agent handoffs.

---

## 🧪 BENCHMARK TASK
- **Input**: "Build a research-to-article workflow using 3 agents."
- **Output**: Researcher -> Writer -> Reviewer -> Publication plan.
