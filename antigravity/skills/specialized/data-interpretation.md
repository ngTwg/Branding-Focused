# 📈 Skill: Data Interpretation

> **PURPOSE:** Convert raw data (JSON, CSV, SQL) into actionable insights.
> **CATEGORY:** Specialized
> **TIER:** 2+

---

## 🚦 ACTIVATION SIGNALS
- Keywords: `analyze data`, `what does this CSV mean?`, `derive insights`, `calculate stats`.
- Task: Performance profiling results, user behavior data, error logs analysis.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Context Mapping**: Identify types, ranges, and unit definitions in raw data.
2. **Standardization**: Clean or normalize data if dirty (handling nulls, outliers).
3. **Hypothesis vs Truth**: Enumerate what the data "suggests" vs "proves".
4. **Statistical Summarization**: Calculate min, max, avg, and median (if applicable).
5. **Insights Synthesis**: Draw 3-5 high-level conclusions for the user.

---

## 📝 OUTPUT SIGNATURE
- Data Summary Table.
- "So-What" Insights Section.
- Recommendation list based on data.
- (Optional) Visualization (ASCII or Mermaid) for trends.

---

## 🧪 BENCHMARK TASK
- **Input**: "Analyze these 50 performance logs and find the bottleneck."
- **Output**: Latency p99 -> Correlation with query X -> Fix recommendation.
