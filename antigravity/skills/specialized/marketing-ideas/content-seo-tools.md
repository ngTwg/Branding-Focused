# Content, SEO & Free Tools (Growth Ideas)

### Search & Content Strategies
- **Easy Keyword Ranking**: Target low-competition, niche, long-tail variations.
- **Glossary Marketing**: Define industry terms to capture "what is X" searches.
- **Programmatic SEO**: Build template-driven pages at scale (e.g., "X for Y").
- **Proprietary Data Content**: Leverage product data for original research.
- **Content Refreshing**: Update old posts with new data to protect rankings.
- **Knowledge Base SEO**: Optimize support docs for problem-solution queries.
- **Parasite SEO**: Publish on high-authority platforms (Medium, Substack) to rank faster.

### Engineering as Marketing (Free Tools)
- **Side Projects**: Build small, useful tools related to your main category.
- **Calculators**: ROI calculators, pricing estimators, or savings tools.
- **Scanners/Checkers**: Tools that audit something for users (e.g., security checker).
- **Chrome Extensions**: Provide standalone value through browser plugins.
- **Importers**: Build easy data migration tools from competitors.
- **Microsites**: Focused domains for specific campaigns or tools.

### Competitor Tactics
- **Comparison Pages**: "[Product] vs [Competitor]" capturing high-intent searchers.
- **Alternatives Pages**: Targeting "[Competitor] alternatives" keywords.
- **Marketing Jiu-Jitsu**: Turn competitor weaknesses (price hikes, outages) into your hooks.
