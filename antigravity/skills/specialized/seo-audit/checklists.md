# SEO Audit Checklists & Scoring System

Use these checklists to perform a comprehensive SEO health check across your site or specific pages.

---

## Technical SEO Audit Checklist

### Crawlability
- [ ] Robots.txt exists and is configured correctly.
- [ ] XML Sitemap exists and is submitted to Search Console.
- [ ] Sitemap contains only canonical, indexable URLs.
- [ ] No unintentional blocks on important pages.

### Indexation
- [ ] site:domain.com check yields the expected number of pages.
- [ ] Search Console coverage report is free of major errors.
- [ ] All pages have self-referencing canonical tags.
- [ ] HTTP → HTTPS canonicals are consistent.

### Core Web Vitals
- [ ] LCP (Largest Contentful Paint) < 2.5s.
- [ ] INP (Interaction to Next Paint) < 200ms.
- [ ] CLS (Cumulative Layout Shift) < 0.1.
- [ ] No layout shift issues during page load.

---

## On-Page SEO Audit Checklist

### Title Tags
- [ ] Every page has a unique title.
- [ ] Primary keyword appears in the first half of the title.
- [ ] Title length is 50-60 characters.
- [ ] Title includes a compelling hook or benefit to the reader.

### Meta Descriptions
- [ ] Every page has a unique meta description.
- [ ] Description length is 150-160 characters.
- [ ] Starts with an action word (Learn, Understand, Discover).
- [ ] Promises specific value or lists what the reader will learn.

### Heading Structure
- [ ] Only one H1 per page (usually the title).
- [ ] H1 contains the primary keyword.
- [ ] All secondary headings follow a logical hierarchy (H2 → H3 → H4).

---

## Internal Linking Checklist

- [ ] Every indexable page is reachable within 3 clicks of the homepage.
- [ ] Links use descriptive anchor text (not "click here" or "this article").
- [ ] Every page has at least 3-5 internal links pointing to relevant related pages.
- [ ] Breadcrumbs are implemented for clear navigation through categories.

---

## Content Quality Checklist (E-E-A-T)

- [ ] Experience: Content demonstrates first-hand knowledge or original data.
- [ ] Expertise: Author bio and credentials are visible and accurate.
- [ ] Authoritativeness: Content is cited by other reputable sites in the space.
- [ ] Trustworthiness: Site is secure (HTTPS), has contact information, and clear policies.
- [ ] Word count: At least 1,500 words for primary informational keywords.
- [ ] Uniqueness: Content is significantly better or more comprehensive than top competitors.
