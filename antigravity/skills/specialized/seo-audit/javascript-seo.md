# JavaScript SEO Audit for Concept Pages

A specialized 30-point audit framework specifically for technical JavaScript concept pages (e.g., Closures, Promises, Event Loop).

---

## 1. Title Tag Checklist (4 points)

- [ ] Length: 50-60 characters.
- [ ] Primary keyword in first half (e.g., "Closures: ...").
- [ ] Ends with "in JavaScript" (standard pattern).
- [ ] Contains a compelling hook (promises value/benefit).

**Title Formula**: `[Concept]: [What You'll Understand] in JavaScript`

---

## 2. Meta Description Checklist (4 points)

- [ ] Length: 150-160 characters.
- [ ] Starts with an action word (Learn, Understand, Discover).
- [ ] Contains primary keyword + "JavaScript".
- [ ] Promises specific value (lists topics covered).

**Description Formula**: `[Action word] [what it is] in JavaScript. [Specific things they'll learn]: [topic 1], [topic 2], and [topic 3].`

---

## 3. Keyword Placement Checklist (5 points)

- [ ] Primary keyword in frontmatter `title`.
- [ ] Primary keyword in frontmatter `description`.
- [ ] Primary keyword in first 100 words of content.
- [ ] Keyword in at least one H2 heading (`##`).
- [ ] Natural reading (no keyword stuffing).

---

## 4. Content Structure Checklist (6 points)

- [ ] Opens with an engaging question hook (How does...? Why do...?).
- [ ] Simple code example appears in the first 200 words.
- [ ] `<Info>` box with "What you'll learn" after the opening.
- [ ] Paragraphs are short (2-4 sentences max).
- [ ] Total word count is 1,500+.
- [ ] Key terms are **bolded** on first mention.

---

## 5. Featured Snippet Checklist (4 points)

- [ ] "What is X" has a concise 40-60 word definition paragraph.
- [ ] At least one H2 is phrased as a question.
- [ ] "How to" content uses `<Steps>` component or numbered list.
- [ ] "X vs Y" sections include a comparison table.

---

## 6. Internal Linking Checklist (4 points)

- [ ] 3-5 related concepts linked within the body (descriptive anchor text).
- [ ] No "click here", "here", or "this article" anchors.
- [ ] Prerequisites linked at early in a `<Warning>` box.
- [ ] "Related Concepts" section at end has 4 cards.

---

## 7. Technical SEO Checklist (3 points)

- [ ] Single H1 per page (only the primary title).
- [ ] URL slug contains the primary keyword only (lowercase, hyphens).
- [ ] Page is not an "orphan" (linked from at least one other page).

---

## Scoring & Status

- **27-30/30**: ✅ Excellent - Ready to publish.
- **23-26/30**: ⚠️ Good - Minor optimizations needed.
- **17-22/30**: ⚠️ Fair - Several improvements needed.
- **0-16/30**: ❌ Poor - Significant work required.
