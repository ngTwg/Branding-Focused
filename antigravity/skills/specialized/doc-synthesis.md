# 📑 Skill: Document Synthesis

> **PURPOSE:** Analyze and synthesize information from multiple long documents.
> **CATEGORY:** Specialized
> **TIER:** 3+

---

## 🚦 ACTIVATION SIGNALS

- Keywords: `synthesize`, `summarize multiple documents`, `find differences`, `literature review`.
- Task: Comparing multiple source files, extracting key insights from long text, or creating a summary.

---

## 🛠️ SUB-STEPS (EXECUTION PIPELINE)

1. **Information Extraction**: Identify main claims, entities, and relationships in each source.
2. **Context Mapping**: Map how information correlates across different documents (Agreements/Conflicts).
3. **Thematic Organization**: Group findings by theme/dimension rather than by source document.
4. **Insight Synthesis**: Condense information into actionable conclusions, identifying gaps or novel points.
5. **Quality Review**: Ensure no hallucinated citations or fake references; use direct quotes where critical.

---

## 📝 OUTPUT SIGNATURE

- Synthesized report (Executive Summary -> Detailed Themes -> Conclusion).
- Comparison table (Themes x Sources).
- Actionable decision framework.

---

## 🧪 BENCHMARK TASK

- **Input**: "Compare these 3 research papers on LLM optimization."
- **Output**: Summary table (Method, Latency, Accuracy) -> Unified conclusion on the best approach.
